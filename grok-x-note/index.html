Xのアルゴリズムの最大の目的は**「ユーザーが長くアプリに滞在して、もっと見たくなるようにする」**こと。

2026

最初の30分〜1時間のエンゲージメント速度が超重要。早く反応（特にリプライ・ブックマーク・RT）がつくと一気に広がる。
  • リプライが特に強い（いいねより重みが高い傾向）。
  • 滞在時間：じっくり読まれる・動画を最後まで見られる投稿が強い。
  • 投稿の質：オリジナルで、意外性や洞察があるもの。繰り返し・コピペ・低品質は不利。
  • 外部リンク：最初の投稿に入れるとリーチがかなり落ちる（ユーザーをX外に逃がしたくないから）。
  • 画像・動画・スレッド：テキストだけより有利。
    画像、動画のオリジナル vs 非オリジナル（アルゴリズムの判断基準）
オリジナル（優遇されやすい）
  • 自分で撮った・作ったリアルな写真
  • 自分に関連するスクリーンショット（自分のGrokとの会話、メモ、手書きメモなど）
  • 自分で撮影・編集した動画（短いネイティブ動画、特に15〜60秒以内）
  • 自分の体験を反映した画像（例：手書きメモ＋自分のデスク写真）
  • 価値を追加したもの（注釈を入れたり、Before/After、データビジュアル、自分の視点で加工したもの）
→ これらは滞在時間が増えやすく、おすすめで有利。オリジナル写真はストック画像より明確に優遇される傾向がある。

非オリジナル（不利になりやすい）
  • ストック写真・フリー画像（ネットから拾ってきた一般的なビジネス写真、風景など）
  • 他人の投稿からダウンロード・再利用した画像・動画
  • 同じ画像・動画を繰り返し投稿（自分でも、微妙に変えてもduplicate detectionで引っかかりやすい）
  • AI生成画像を多用しすぎ（特に無加工でそのまま出すと「生成物っぽい」と判断されやすい場合あり）
  • 水印が入った画像や、他プラットフォームから持ってきた痕跡があるもの
→ これらは低品質シグナルとして扱われやすく、おすすめでのリーチが落ちやすい。重複コンテンツはスパムフィルタにかかるリスクも。

  • アカウントの信頼性：一貫した投稿、健全な会話、作者の評価も影響。

フォロワー数は直接ほとんど関係ない。質の良い投稿が似た興味の人に届きやすい仕組みになってる。

友人のポストに3行くらいの自然なリプライや引用RTをするのは有効。会話が生まれるとアルゴリズムが喜ぶ。

アルゴリズムのスコアリングの詳細

全体の流れ（3段階）
  1. 候補集め：約1,500件のポストを集める（フォロー中＋おすすめ）。
  2. スコアリング（Heavy Ranker / Phoenix）：ここが本番。各ポストに最終スコアを付ける。
  3. 並び替え＋フィルタ：スコアが高い順に表示。スパムや低品質は除外。

主なエンゲージメントの重み（目安：いいね＝1xとして）
これが2026年の大まかな比率（オープンソース解析や実践者まとめから）：

特に重視されるポイント
  • 初期速度（Engagement Velocity）：投稿後30〜60分以内の反応がめっちゃ大事。早い反応があると一気にブースト。
  • 時間減衰（Time Decay）：投稿から時間が経つとスコアが急激に落ちる（6時間で半減とか言われる）。
  • 作者の信頼性・多様性：同じ作者のポストばかりだと減点気味。
  • オリジナル性・メディア：自分で撮った画像・動画、リアル体験が強い。

リプライの重みが

13.5

〜

27x

と幅がある理由


主な理由は2つ
1. 分析ソースや時期による違い
  • Xのアルゴリズム（PhoenixというGrokベースのモデル）はオープンソース化されているけど、公式に「これが正確な倍率です」と全部公開しているわけじゃない。
  • 開発者や分析者がコードを解析して出した数字にバラツキが出る。
    • 古めの解析や基本的な直接リプライ → 13.5x くらい
    • 2025〜2026年の最新モデル（会話重視にシフト） → 27x くらいと評価されるケース
要するに「解析する人やバージョンによって少し数字が変わる」から範囲で表現されることが多いんだ。
2. リプライの「質」によって重みが大きく変わる（これが本質）
  • 普通のリプライ（ただの返信）：13.5x〜27xくらい
  • 作者がそのリプライに返事をする（会話が続く）：75x〜150x（likeの数十〜百数十倍！）
これが一番デカい違い。
単に「リプライがつく」だけじゃなく、**「リプライに対して作者が反応して会話が生まれる」**と、アルゴリズムが「このポストは価値がある、もっと広げよう」と判断してスコアが跳ね上がる。

アルゴリズムは「会話が続くかどうか」をめっちゃ重視してる

Xアルゴリズムの最新アップデート情報（2026年6月現在）

1. 最大の変更：Phoenix（GrokベースのTransformerモデル）本格導入
・5月15日に大規模アップデート（187ファイル変更、18,000行超のコード追加）。
・Grok-1由来のTransformerモデルで各投稿を深く理解・予測するようになった。
・投稿の内容をAIが読み込んで「このユーザーが本当に興味持ちそうか」を予測する。

2. 2026年の主な重視ポイント（変わったところ）
  • 会話の質が最優先：リプライ（特に作者が返事して会話が続く場合）の重みが非常に高い（27x前後、会話継続でさらに大幅ブースト）。
  • 最初の30分（Early Velocity）：投稿直後の反応速度が超重要。遅いとほぼ伸びない。
  • オリジナルコンテンツ：自分で撮った画像・動画、リアル体験談が強い。ストック画像や再利用は不利。
  • 滞在時間（Dwell Time）：じっくり読まれる・動画を最後まで見られる投稿が優遇。
  • ハッシュタグ：ほぼ効果なし。AIがセマンティック理解（意味理解）で分類するようになった。
  • 外部リンク：メイン投稿に入れるとリーチが大幅ダウン（50%減くらい）。リプライに置くのが鉄則。
  • スレッド：単発より3倍くらい伸びやすい傾向。
  • 一貫性：毎日同じ時間に投稿するとアルゴリズムが好む（数日休むと一時的にリーチ抑制）。

3. 最近の細かい変更（2026年）
  • Reply Downvotes（3月頃）：リプライに「thumbs down」が追加され、質の低い返信が抑制される。
  • Followingフィードも一部アルゴリズム化（完全時系列ではなくGrokで関連性考慮）。
  • Premiumユーザー優遇：圧縮緩和やリーチブーストが継続。
  • 動画：ネイティブ動画の重みが増加。

アルゴリズムは今も毎月のように微調整されてる（GitHubのxai-org/x-algorithmで最新コードが見られる）。
基本は「人が止まって読んで、反応したくなる本物の投稿」をAIが推す方向にどんどん進化してる

アルゴリズムが好む投稿時間帯の最適化

アルゴリズムが好む投稿時間帯の最適化とは、投稿直後の初速エンゲージメント（いいね・コメント・保存・シェア・視聴時間）を最大化し、アルゴリズムに「価値ある投稿」と評価されるタイミングを選ぶことです。
SNSのアルゴリズム（Instagram、X/Twitter、TikTokなど）は基本的に「新鮮さ」と「初期反応」を重視します。フォロワーがアクティブな時間に投稿すると、タイムライン上位に表示されやすく、拡散が加速します。逆にオフライン時間帯に投稿すると埋もれやすいです。




  • 平日（月〜木）：朝7-9時や夜20-22時が特に強い。
  • 金曜：夜遅め（22時〜）もOK。
  • 週末（土日）：朝10-11時や午後16-18時。
X（Twitter）特化：平日朝9時前後（特に火・水曜日）が強い。中午〜午後（12-18時）も会話が活発。
Instagram：木曜9時、水曜12時・18時あたりがデータで上位。Reelsは夜も良い。
最適化の具体的な方法（PDCA）
  1. インサイトを確認
    • Instagram/TikTok：フォロワーの「最もアクティブな時間」をAnalyticsでチェック。
    • X：アナリティクスでインプレッションが高い時間を分析。
  2. A/Bテストを実施
    • 同じ内容を違う時間帯（例：12時 vs 21時）に投稿して比較。
    • 2-3週間続けて傾向を掴む。
  3. 頻度と組み合わせ
    • 毎日1-3投稿程度（多すぎると逆効果）。
    • シリーズ化や固定ハッシュタグで継続視聴を促す。
    • ストーリーズで投稿告知して初速をブースト。
  4. ツール活用
    • スケジューラー（Buffer、Hootsuiteなど）で自動投稿。
    • 分析ツールでデータ駆動。
注意点
  • ターゲット次第：学生向け→夜、ビジネス→朝・昼、趣味→夜・週末。
  • コンテンツ品質が最優先：時間帯が良くても内容が弱いとアルゴリズムは評価しない。視聴時間や保存率を意識。
  • テストを繰り返す：アルゴリズムは更新されるので、定期的に見直しを。
