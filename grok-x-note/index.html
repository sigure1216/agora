NOTE — X GROWTH

Grokに聞いた、
Xを伸ばす方法

Xのアルゴリズムがいちばん大事にしているのは、「ユーザーがアプリに長く滞在し、もっと見たくなること」。その一点を軸に、2026年時点でわかっていることを整理しました。

01

アルゴリズムが特に重視すること（2026年）

投稿してから最初の30分〜1時間の反応速度が、いちばん効きます。早く反応（特にリプライ・ブックマーク・リポスト）がつくと、一気に広がります。

  • リプライがいちばん強い（いいねより重みが大きい傾向）。
  • 滞在時間：じっくり読まれる・最後まで見られる投稿が伸びる。
  • 投稿の質：オリジナルで、意外性や洞察があるもの。繰り返し・コピペ・低品質は不利。
  • 外部リンク：最初の投稿に入れるとリーチが大きく落ちる（X外へ逃がしたくないため）。
  • 画像・動画・スレッド：テキストだけより有利。

画像・動画は「オリジナルかどうか」で評価が分かれる

◎ オリジナル（優遇されやすい）

  • 自分で撮った・作ったリアルな写真
  • 自分に関係するスクリーンショット（Grokとの会話、メモ、手書きなど）
  • 自分で撮影・編集した短い動画（特に15〜60秒）
  • 体験を反映した画像（手書きメモ＋デスクの写真 など）
  • 価値を足したもの（注釈、Before/After、データの可視化）

滞在時間が伸びやすく、おすすめで有利。オリジナル写真はストック画像より明確に優遇されます。

△ 非オリジナル（不利になりやすい）

  • ストック写真・フリー画像（一般的なビジネス写真や風景）
  • 他人の投稿から再利用した画像・動画
  • 同じ画像の繰り返し（少し変えても重複判定にかかりやすい）
  • AI生成画像の多用（無加工だと「生成物っぽい」と判断されやすい）
  • 透かし入りや、他プラットフォームの痕跡が残るもの

低品質シグナルとして扱われ、リーチが落ちやすい。重複はスパム判定のリスクも。

フォロワー数は、ほとんど関係ありません。質の良い投稿が、近い興味を持つ人へ届く仕組みです。

  • アカウントの信頼性：一貫した投稿、健全な会話、作者の評価も影響します。
  • 友人の投稿に3行くらいの自然なリプライや引用リポストをするのは有効。会話が生まれるとアルゴリズムが好みます。
  • 毎日同じ時間に1〜2投稿し、反応には素早く返すのがコツです。
02

おすすめに乗りやすい、具体的なテクニック

おすすめは「この投稿を、この人が見たら反応しそうか？」をAIが予測して決めます。だから、早い反応（最初の30〜60分）と、会話が生まれる内容がいちばん大事です。

  1. 冒頭（フック）を強くする。共感＋意外性で、スクロールを止める。ここが超重要。
  2. 3〜5行で簡潔に＋質問で締める。長文は読み飛ばされる。最後は質問形で反応を誘う。
  3. ビジュアルを必ず入れる（約2倍有利）。テキストだけより滞在時間が伸びる。
  4. スレッドを活用する。単発より3倍ほど伸びやすい傾向。
  5. バズっている友人の投稿に乗る。引用リポストで自分の視点を短く（3〜5行）、投稿から1時間以内に。
  6. その他の細かいコツ。
    • 投稿時間：朝の通勤（7〜9時）、昼休み（12時頃）、夜（19〜22時）
    • 外部リンクは本文に入れず、最初のリプライに置く
    • ハッシュタグはほぼ効果なし。無理に入れない
    • 一貫性：週3〜5回、同じテーマで投稿。「このアカウントは○○専門」と認識される
    • 反応返し：リプライが来たら最初の1〜2時間で必ず返す（さらにブーストがかかる）
03

スコアリングの仕組み

表示順は、ざっくり3段階で決まります。

  • 候補集め：約1,500件の投稿を集める（フォロー中＋おすすめ）。
  • スコアリング（Heavy Ranker / Phoenix）：ここが本番。各投稿に最終スコアを付ける。
  • 並び替え＋フィルタ：高い順に表示。スパムや低品質は除外。

エンゲージメントの重み（いいね＝1x を基準に）

シグナル重みの目安
いいね1x（基準）
リプライ13.5〜27x
リプライに作者が返信し、会話が続く75〜150x

※ リポスト・ブックマークも強いシグナルとして扱われます。

特に効くポイント

  • 初期速度：投稿後30〜60分の反応が最重要。早い反応で一気にブースト。
  • 時間減衰：時間が経つとスコアが急落（6時間で半減とも）。
  • 作者の多様性：同じ作者の投稿ばかりだと減点気味。
  • オリジナル性・メディア：自分で撮った画像・動画、リアルな体験が強い。
04

なぜリプライの重みは「13.5〜27x」と幅があるのか

1. 解析ソースや時期の違い

  • Xのアルゴリズム（GrokベースのPhoenix）は公開されているが、公式が正確な倍率を全部出しているわけではない。
  • 解析者によって数字にばらつきが出る。
  • 古い解析・基本的な直接リプライ → 13.5x ほど。
  • 2025〜2026年の最新モデル（会話重視）→ 27x ほど。

2. リプライの「質」で重みが大きく変わる（ここが本質）

  • 普通のリプライ：13.5〜27x
  • 作者が返信して会話が続く：75〜150x（いいねの数十〜百倍以上）

「リプライがつく」だけでなく、「作者が反応して会話が生まれる」と、アルゴリズムは“価値がある”と判断してスコアが跳ね上がります。会話が続くかどうかを、とても重視しています。

05

最新アップデート（2026年6月時点）

最大の変更：Phoenix（Grokベースのモデル）本格導入

  • 5月15日に大規模アップデート（187ファイル・18,000行超）。
  • 投稿内容をAIが深く理解し、「この人が本当に興味を持つか」を予測するようになった。

2026年に効くポイント（変わったところ）

  • 会話の質が最優先（作者が返信して続く会話の重みが非常に高い）。
  • 最初の30分の反応速度が超重要。遅いとほぼ伸びない。
  • オリジナルコンテンツが強い。ストック・再利用は不利。
  • 滞在時間が長い投稿を優遇。
  • ハッシュタグはほぼ無効（AIが意味で分類）。
  • 外部リンクは本文に入れるとリーチ半減ほど。リプライに置くのが鉄則。
  • スレッドは単発より3倍ほど伸びやすい。
  • 一貫性：毎日同じ時間が好まれる（数日休むと一時的に抑制）。

最近の細かい変更

  • リプライに低評価（thumbs down）が追加され、質の低い返信を抑制（3月頃）。
  • フォロー中フィードも一部アルゴリズム化（完全な時系列ではない）。
  • Premiumユーザーは優遇が継続。
  • ネイティブ動画の重みが増加。

毎月のように微調整されています（最新コードは GitHub の xai-org/x-algorithm）。基本は「人が止まって読み、反応したくなる本物の投稿」を推す方向へ進化しています。

06

投稿時間帯の最適化

ねらいは、投稿直後の初速（いいね・コメント・保存・シェア・視聴時間）を最大化できるタイミングを選ぶこと。アルゴリズムは「新鮮さ」と「初期反応」を重視するので、フォロワーが活発な時間に出すと上位に表示されやすくなります。

曜日と時間帯の目安

曜日おすすめ時間帯ひとこと
平日（月〜木）7〜9時 ／ 20〜22時朝の通勤・夜のくつろぎ時間
金曜22時〜夜遅めもよい
週末（土・日）10〜11時 ／ 16〜18時朝と夕方
X特化平日 朝9時前後（火・水）昼〜午後（12〜18時）も会話が活発

見直しの手順（PDCA）

  • インサイトを確認：アナリティクスでインプレッションの高い時間を分析。
  • A/Bテスト：同じ内容を違う時間に出して比較。2〜3週間続けて傾向を掴む。
  • 頻度：毎日1〜3投稿（多すぎは逆効果）。
  • ツール活用：スケジューラーで自動化し、データで判断。

時間帯が良くても、内容の質が最優先です。アルゴリズムは更新され続けるので、定期的に見直しを。
